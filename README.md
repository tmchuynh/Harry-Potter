# Harry-Potter

## The [HP-API](https://hp-api.onrender.com/)

This api has routes to request data on the characters (specifically the students, staff, and houses) and spells

## Characters
### What's provided
- Character name
- Actor
- House
- Wand information
- Ancestory information
- Student (True/False)
- Staff (True/False)
and much more!

## Spells
### What's provided
- Name of the spell
- Functionality
